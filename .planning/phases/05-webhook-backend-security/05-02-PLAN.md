<objective>
Add rate limiting (slowapi), CORS environment variable restriction, and bearer token authentication to all endpoints.

Purpose: Security hardening prevents toll fraud (runaway dialer), blocks unauthorized API access, and restricts CORS to the dashboard origin. Without rate limiting, a bug in the auto-dialer could burn through call credits. Without auth, anyone with the URL could trigger calls.

Output: main.py with slowapi rate limits on all endpoints, bearer token auth on initiate-call and dashboard endpoints, and CORS restricted to DASHBOARD_ORIGIN env var.
</objective>

Create the bearer scheme and verification function AFTER the DASHBOARD_SECRET_KEY line:
```python
bearer_scheme = HTTPBearer(auto_error=True)

async def verify_bearer_token(
    credentials: HTTPAuthorizationCredentials = Depends(bearer_scheme),
) -> str:
    """Validate bearer token matches DASHBOARD_SECRET_KEY."""
    if not DASHBOARD_SECRET_KEY:
        raise HTTPException(status_code=500, detail="Server misconfigured: no dashboard secret key")
    if credentials.credentials != DASHBOARD_SECRET_KEY:
        raise HTTPException(status_code=401, detail="Invalid token")
    return credentials.credentials
```
